# 二叉树  

### 主要介绍二叉树和哈夫曼树(霍夫曼树)  
树是非线性数据结构,它能很好的描述数据的层次关系.类似我们的文件夹分级,就是树结构  
二叉树是最长用的树,为父节点带两个儿子,左儿子和右儿子,二叉树应用广泛,是递归和搜索实现的基础数据结构  
主要讲解二叉树的性质和储存,还有遍历,以及遍历关系(搜索见3章节)  
二叉树的遍历有:前序遍历,后序遍历,中序遍历和层序遍历,如果已知中序遍历和其他任何一种遍历就能构建出唯一二叉树  

哈夫曼树是一类带权路径最短的最优树,是贪心思想在二叉树上的应用  
对于哈夫曼树其实不难,但是是考点,而且竞赛中有考到相关变种,主要考其中的贪心思想
