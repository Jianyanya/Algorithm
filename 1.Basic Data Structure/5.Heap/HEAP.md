# 堆  

### 主要介绍二叉树堆和优先队列
堆是一种树形结构,树的根是堆顶,堆顶始终保持为所有元素的最优值.  
有最大堆和最小堆,最大堆则是根节点是最大值,最小堆相反.  
堆一般用二叉树实现,称为二叉堆,二叉堆是一颗完全二叉树(除最后一层外,所有层都完全填满,并且最后一层的节点都尽可能地向左靠拢)  
二叉堆的应用为对顶堆(动态维护中位数/第K大数的利器),是本章的重点,除此之外还有[配对堆,左偏树,二顶堆,斐波那契堆](https://oi-wiki.org/ds/heap/)  
由于完全二叉树的性质,我们可以用数组来存储一个二叉堆,非常好用  

优先队列是一种抽象数据类型(ADT),堆是实现优先队列这种ADT的高效数据结构,除了堆,还可以使用无序数组和有序数组来实现,但通常堆(删插取平衡效率)比较高效,所以默认为堆实现
